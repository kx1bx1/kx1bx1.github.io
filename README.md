# kx1bx1.github.io

A curated extension gallery published via GitHub Pages.

This repository contains the source, build tooling, and workflows used to publish the gallery site. Community contributions are welcome.

---

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Install

```bash
npm ci
```

### Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port shown in your terminal). It should automatically copy the address to your clipboard.

### Build

```bash
npm run build
```

### Lint & format

```bash
npm run lint  # or lint:fix to automatically fix some errors
npm run format
```
